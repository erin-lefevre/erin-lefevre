Seafloor: Raster Processing Toolset

The Seafloor toolset combines common geoprocessing tools into a powerful and easy to use standalone python application that is intended to reduce production time, increase consistency, and to provide an application that anyone can use with minimal knowledge or the need for a complete Geographic Information System. The toolset is open-source and can be customized with Python 3.6 and PyQt 5. It was designed to be useful to a wide range of users including scientists who may prefer an open-source dedicated toolset to amateurs looking to generate their first hiking map
